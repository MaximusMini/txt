Yii2 - установка зависимостей через composer

	Проверка наличия composer
		composer
		
	Обновление composer
		composer self-update
		
	Установка зависимостей
		composer install
		
	!!!
		установку зависимостей нужно запускать из корня проекта
	
	!!!
		проверка версии php
			php -v
	
	!!!
		при необходимости нужно сменить версию php (в настройках OServera) и полностью его перезагрузить
