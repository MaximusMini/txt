1.Создаем новый шаблон basic.php
	
	папка view/layouts используется для шаблонов приложения
	шаблон main.php используется по умолчанию
	создадим в папке view/layouts новый шаблон basic.php
	
			<!DOCTYPE html>
			<html>
			<head>
				<meta name="viewport" content="width=device-width, initial-scale=1">
				<title>Шаблон basic</title>
			</head>
			<body>
			<p>Верхняя часть шаблона</p>
			<!-- Верхняя часть остается неизменной при любыхдействиях использующих этот шаблон-->

			<!-- Средняя часть - выводятся разные виды (представления) приложения, в зависимости от действий-->
			<?= $content ?>

			<p>Нижняя часть шаблона</p>
			<!-- Нижняя часть остается неизменной при любыхдействиях использующих этот шаблон-->
			</body>
			</html>
	
2.Генерируемый новый контроллер
	
	для работы приложения нужно создать контроллер, который будет управлять действиями пользователя
	в данном случае при заходе на сайт выводить "Привет, МИР!"
	
	переходим в генератор кода gii
	http://myyii/web/index.php?r=gii
	
	раздел Controller Generator -> Start
	
	поле Controller Class - название нового контроллера вместе с путем где он будет расположен
	app\controllers\BasicController
	смотрим какие файлы буду созданы -> нажимаем снизу Preview
	controllers\BasicController.php
	views\basic\index.php
	
	Action IDs - это действие, которое будет добавлено в контроллер
	через запятую можно добавить несколько действий
	в нашем случае в контроллере будет одно действие index
	
	BaseClass - указывает на yii\web\Controller - это класс контроллеров
	
	в результате генерации будут созданы
		контроллер BasicController.php в папке controllers
		файл index.php представление для действия index контроллера BasicController
	
	файлы представления и действия контроллера называются одинаково
	
	нажимаем Generate
	
3.Настройка приложения

	добавить созданный шаблон (basic.php) и контроллер (BasicController) в настройки приложения для использования их по умолчанию
	
	открыть файл config/web.php
	
	добавляем 'layout' => 'basic',
	это означает, что приложение будет использовать созданный шаблон views\layouts\basic.php по умолчанию
	
	добавляем 'defaultRoute' => 'basic/index'
	она означает, что приложение будет использовать созданный нами контроллер (BasicController) и 
	добавленное в него действие actionIndex по умолчанию
	


5.Настраиваем действие actionIndex созданного контроллера BasicController

	прописываем в действии переменную
	$hello = 'Привет мир';
	
	чтобы передать переменные и объекты в представление (view) нужно в метод render(),
	после названия представления (первый параметр метода render), в виде массива добавить нужные
	переменные и объекты
	
	render('index',['viewHello'=>$hello])
	это строка означает, что в представление 'views/basic/index',
	передается переменная $hello, которая будет доступна там,
	как переменная $viewHello 
	
6.Настройка созданного шаблона
	
	указываем (в секции <html>), чтобы шаблон использовал язык и местоположение, определенное в настройках
	<html lang="<?=Yii::$app->language?>">
	
	добавляем необходимые настройки приложения
	открыть файл config/web.php
	
	для языковых настроек добавляем строку (после 'defaultRoute' => 'basic/index')
	'language'=> 'ru_RU',
	ru_RU - обозначает,что приложение использует русский язык
	
	указываем, чтобы шаблон использывал кодировку приложения 
	<meta charset='<?=Yii::$app->charset?>'>
	
	файл config/web.php
	добавляем
	'charset'=>'UTF-8'
	
	в качестве названия страницы временно используем имя самого приложения
	<title><?=Yii::$app->name?></title>
	
	файл config/web.php
	добавляем
	'name'=>'Приложение',
