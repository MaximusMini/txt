1	Переменные создаются с помощью ключевого слова "var"

2	Правила имени переменной
		2.1) Переменная не может состоять только из одних цифр.
		2.2) Переменная не должна содержать пробельных символов.
		2.3) Переменная не может быть ключевым словом (например, нельзя назвать переменную "var").
		2.4) Переменная не может содержать различные спецсимволы (кавычки, апострофы, восклицательный и вопросительный знаки, точка, запятая и прочее), однако, дефис и знак подчёркивания использовать можно.

3	Переменная инициализируется с помощью знака =
		var x = 5; // создание и инициализация переменной
		
4	Вывод переменной в окно браузера
		document.write(x);
