Стратегия тотал

1	Выбрать матчи с предполагаемой напряженной борьбой
		- матчи принципиальных соперников
		- мотивация команд
		- по критериям личных встреч
			* кол-во заброшенных шайб в личных встречах (тотал - в среднем за матч)
			* кол-во штрафного времени в личных встречах (в среднем за матч)
		- матчи команд, находящихся рядом в турнирной таблице

2	Определить у кого из играющих команд худшее меньшенство
		http://www.khl.ru/stat/teams/468/powerplay-ga/

3	Определить у кого из играющих команд лучшее большинство
		http://www.khl.ru/stat/teams/468/powerplay-gf/

4	Количество заброшенных и пропущенных шайб за последние 5 матчей
		- тотал каждой команды за последние пять матчей
		- тотал за все матчи каждой команды
		
5	Тотал команд дома/в гостях
		
6	Анализ очных встреч команд
		- количество заброшенных шайб за матч (П1 забр. шайбы + П2 забр. шайбы)/ кол-во очных встреч

7	Анализ игры вратарей

8	Посчитать количество игр которые команда забивала 1, 2, 3 и т.д. шайб (за все матчи, за последние пять)	
		это будет более точно чем среднее за все матчи

9	Посчитать тоталы матчей и их количество (всех + последние пять)

10	Количество забрасываемых шайб за матч (среднее + количество матчей)
		дома
		в гостях

Виды тотала

		общий тотал (вкл. от)
		индивидуальный тотал команд
		тотал периодов
		самый результативный период
		
		
Выводы:

	2	эвристический анализ каждого матча с учетом п.1
	

Техзадание - для одного матча
	1 Прогноз формировать путем выбора команд из выпадающего списка
	2 Алгоритм работы
		2.1 определяем как команды играют в большинстве
		2.2 определяем как команды играют в меньшенстве
		
	
	
Техзадание - для матчей дня	
	1 Указываем какие команды играют (выбираем из списка)
	2 Для этих команд формируем таблицы с показателями
		игра в большинстве - кто лучше всего играет в большинстве
			процент реализации большинства
			количество полученных численных преимуществ
			количество шайб заброшенных в большинстве
		игра в меньшинстве - кто хуже всего играет в меньшинстве
			процент реализации большинства соперником
			количество полученных численных преимуществ соперником
			количество шайб пропущенных в меньшинстве
	
	
Алгоритм - 	для матчей дня
	1 Определяем команды для которых нужно вывести данные (через checkbox)
	2 http://www.khl.ru/stat/teams/468/powerplay-gf/
		- код сохраняем в файл powerplay-gf.php
		- парсим данные в многомарный массив (таблицу БД)
							powerplay[id команды][games] - количество игр
							powerplay[id команды][perc_more] -  процент реализации большинства
							powerplay[id команды][perc_less] -  процент реализации большинства соперником
							powerplay[id команды][count_more] - количество полученных большинства
							powerplay[id команды][count_less] - количество полученных большинства соперника
							powerplay[id команды][puck_more] - количество шайб в большинстве
							powerplay[id команды][puck_less] - количество шайб пропущенных в меньшенстве
	3 со страниц команд (champ) парсим данные о результатах прошедших матчей в БД
		- данные заносятся в БД с результатами всех команд
			таблица result_match
				id_team			- id команды
				rival			- соперник по матчу
				place			- место игры дома/в гостях
				date_match		- дата игры
				time_end		- время окончания (основное время, ОТ, Б)
				puck_team		- количество шайб заброшенных командой
				puck_rival		- количество шайб заброшенных соперником
				result			- результат игры для команды (победа - win, проигрыш - lose)
				
			создание таблицы
			CREATE TABLE result_match (id_team INT, name_team VARCHAR(255), rival VARCHAR(255), place VARCHAR(255), date_match DATE, time_end VARCHAR(255), puck_team INT, puck_rival INT, result VARCHAR(255) )
		- интерфейс
			Главная (main.php) -> Тотал (total.php)
	

Рабочий алгоритм
	1	main.php -> record_results.php (запись результатов)
			в папку temp заносятся html-страницы с результатами сыгранных матчей каждой команды
			эти файлы затем нужно перенести в папку result_team :)
	2	main.php -> pars_results.php (парсинг результатов)
			в таблицу result_match заносятся результаты всех сыгранных матчей (из сохраненных файлов)
	3	main.php -> total.php (расчитать тотал команд)
		выбрать команду хозяев | команду гостей
		3.1 total.php -> scripts_php/totalMatch5.php (Тотал за последние 5 матчей  - без учета шайб в ОТ и Б)
			выводить среднее количество заброшенных шайб за последние 5 матчей
			запрос - SELECT * FROM result_match WHERE id_team=1 ORDER BY date_match DESC LIMIT 5
		3.2 total.php -> scripts_php/total_all.php (Тотал за все сыгранные матчи - без учета шайб в ОТ и Б)
			выводить среднее количество заброшенных шайб за все матчи
			запрос - SELECT * FROM result_match WHERE id_team=1
		3.3. total.php -> scripts_php/total_home.php (Тотал дома - без учета шайб в ОТ и Б)	
		
		3.4. total.php -> scripts_php/total_guest.php (Тотал в гостях - без учета шайб в ОТ и Б)
