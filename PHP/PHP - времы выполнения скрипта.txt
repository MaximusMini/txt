PHP - времы выполнения скрипта

1	В php.ini прописать новое значение параметра max_execution_time:
		max_execution_time = 60
		60 измеряется в секундах
			Только это не панацея - через 60 сек некоторые броузеры разрывают соединение, и тогда apache прибивает процесс
			
2	Изменить параметр в  php.ini программно
		ini_set() - устанавливает значение опции конфигурации
			!!! Не все доступные опции могут быть изменены с помощью ini_set()
		// в скрипте указать новое значение параметра max_execution_time
		ini_set("max_execution_time", "60");
			60 - время выполнения скрипта в секундах
			как правило функция вызывается в начале кода
		
3	set_time_limit - ограничивает время выполнения
		Устанавливает время в секундах, в течение которого скрипт может работать.
		При вызове set_time_limit() рестартует счётчик таймаута с нуля
		
4	код для определения времени выполнения скрипта php
		кидаем этот код в начало скрипта
			$start = microtime(true);
		а вот это в самый конец скрипта (округляется до 4 знака после запятой)
			echo 'Время выполнения скрипта: '.round(microtime(true) - $start, 4).' сек.';
