1	Полученную из БД дату перевести в формат времени Unix с помощью метода strtotime():
		$mysqldate = '2011-06-10 15:18:00'; - дата и время из БД
		$time = strtotime($mysqldate);

2	C помощью функции date() вывести нужный формат даты и времени
		$day   = date( 'j',$time ); # С помощью функции date() получаем число дня
		$year  = date( 'Y',$time ); # Получаем год
		$hour  = date( 'G',$time ); # Получаем значение часа
		$min   = date( 'i',$time ); # Получаем минуты

3	Для преобразования месяца в текстовый формат создать дополнительный ассоциативный массив где каждому числу месяца присвоить название месяца
